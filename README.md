Photo Fence
===========

Browse photos that were taken within a given time/location window.


Requirements
------------
* PHP
* Ruby (https://rvm.io/rvm/install)
* Ruby Bundler (http://bundler.io/)


Set-up
------
* Run bundler to install Ruby gem dependencies:

    ``$ bundler install``

* Compile asset files locally

    ``$ cap assets:compile``
